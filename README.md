# Simple AI Assistant

Interactive **Streamlit** app that acts as a **Python AI assistant**.  
Ask questions about Python, and the app provides answers using a Retrieval-Augmented Generation (RAG) system with LangChain and Ollama.

## Features
- Streamlit interface for easy user input
- Load and split text documents into chunks
- Create a vector store using ChromaDB
- Generate answers with an Ollama LLM
- Fast, interactive, and easy to run
